# 🪙 Toss App (Head/Tail)

A simple and fun **Coin Toss Web App** built using HTML, CSS, and JavaScript. This app simulates a real coin flip and randomly displays either **Head** or **Tail** on each button click.

---

## 🚀 Features

* 🎲 **Random Toss Simulation** – Generates a random result (Head or Tail)
* 🖼️ **Dynamic Image Update** – Changes image based on toss result
* 🖱️ **Interactive Button** – User triggers toss with a click
* ⚡ **Instant Feedback** – Result appears immediately

---

## 🧠 Concepts Used

* DOM Manipulation
* Event Handling (Click Events)
* Random Number Generation
* Conditional Logic (if/else)

---

## 📂 Project Structure

* `HTML` → Layout and structure
* `CSS` → Styling and button design
* `JavaScript` → Toss logic and image switching
* `Assets (Pics)` → Head and Tail images

---

## 🎯 How It Works

1. User clicks the **Toss button**
2. A random number is generated
3. Based on the result:

   * **Head image** is shown
   * OR **Tail image** is shown
4. The image updates instantly on the screen
